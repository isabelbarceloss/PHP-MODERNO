# Curso de PHP MODERNO - Curso em Vídeo
Esse repositório concentra todos os exercícios que fiz acompanhando o curso.

## `O que eu aprendi:`
- [x] Preparar o ambiente para rodar PHP (servidor local, configuração, arquivos .php);
- [x] Criar scripts PHP básicos: estrutura de arquivo, saída de dados pra página, sintaxe da linguagem;
- [x] Variáveis, tipos de dados, operadores, manipulação de strings e dados primários com PHP;
- [x] Integração entre HTML e PHP, misturando marcação e lógica para gerar páginas dinâmicas;
- [x] Receber e manipular dados de usuário via formulários ou parâmetros (GET/POST/superglobais);
- [x] Processar lógica básica com PHP (condicionais, cálculos, manipulação de dados) para tornar as páginas interativas e dinâmicas.
